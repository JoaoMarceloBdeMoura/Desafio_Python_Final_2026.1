->Explicação do funcionamento do programa:

Progama que diz se é viável ou não fazer uma viajem, ele converte o valor que foi inserido pelo usuário para Euro, faz cálculos com os dados inseridos pelo usuário sobre: preço da passagem, estadia e quantos dias irá ficar. A pergunta sobre o 
país é meramente decoração


->○ Instruções de uso:

Entre com o valor que está pretendendo gastar, o preço da passagem e o tempo de estadia, o algorítimo irá responder se a viajem é possível e viável ou não.

■ Qual a diferença entre o comando git add . e git commit -m "mensagem"?

R: O git add . adicona todas as alterações feitas e o git commit -m"" faz um commit local e nomeia o commit

■ Por que é necessário realizar o casting (conversão de tipo) ao usar a função input() em Python para cálculos matemáticos?

R: Porque o input() recebe dados como strings e cálculos matemáticos com strings podem dar erro ou resuldados não esperados

■ O que acontece se tentarmos somar uma variável do tipo str com uma do
tipo float?

R: Acontece um TypeError
